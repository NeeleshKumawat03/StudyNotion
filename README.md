# StudyNotion


## Usage Instruction

1. Install the required packages.
    ```sh
    npm install
    ```

1. Start the development server.
    ```sh
    npm start
    ```
1. Open the project in your browser at [`http://localhost:3000`](http://localhost:3000) to view your project.
